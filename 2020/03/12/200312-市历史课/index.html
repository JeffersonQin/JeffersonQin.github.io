第二课 古代世界的帝国与文明的交流

古代文明的扩展

最初的文明：亚非欧大陆的若干地区，但是之间联系比较少

扩展的条件 / 原因

  • 农耕文明

    • 比较发达的社会分工
    • 相对较高的劳动生产率
    • 复杂的社会组织和管理系统

  • 海洋文明

    • 组织能力
    • 航海技术
    • 武器

各个文明具体的扩展

  • 古埃及文明

    • -> 叙利亚、巴勒斯坦
    • 埃及新王国时期：两河流域

  • 西亚文明

    • -> 两河流域
    • 古巴比伦王国首次实现了统一 & -> 地中海东岸

  • 亚述

    • ->两河流域地区
      -> 小亚细亚的一部分
      -> 一度征服埃及

  • 古希腊

    • 移民 -> 地中海、黑海周边地区殖民
      建立众多城邦国家

文明的交流

农耕技术

  • 西亚 -> 中亚，欧洲，北非

冶铁技术

  • 西亚 -> 埃及，希腊

雕刻艺术

  • 埃及 -> 希腊

字母文字

  • 西亚腓尼基

    • 西亚文明的交汇点
    • 商业比较发达
    • 方便记录 -> 22个字母 ”腓尼基字母“

    • 衍化

      • 东方：阿拉玛字母

        • 古代西亚
        • 埃及
        • 印度

      • 向西：希腊

        • 希腊字母

          • 拉丁字母

            • 欧洲所有字母文字

丝绸之路

  • 罗马（大秦）与汉

    • 东汉班超，派甘英出使
    • 间接经贸文化交流

    • 罗马政府地中海东部后

      • 与汉朝直接建立关系
      • 2世纪，通商

古代世界的帝国

亚述（不是帝国）

  • 时间：公元前八世纪

  • 跨度：亚非

埃及新王国（不是帝国）

  • 跨度：非亚

波斯帝国

  • 时间：公元前六世纪
  • 兴起于：伊朗高原
  • 跨度：亚非欧
  • 主要君主：大流士一世

  • 制度：君主专制制度

    • 国王：核心、最高主宰
    • 君权神授

    • 地方：行省制

      • 行省总督 & 军事长官 互相监督制约

    • 较完善的官僚体系、税收系统

    • 波斯人担任重要职务

亚历山大帝国

  • 时间：公元前四世纪晚期

  • 兴起

    • 原因：希波战争两败俱伤
    • 初始：马其顿国王亚历山大率军入侵波斯帝国
    • 过程：10年，灭波斯

  • 跨度：亚非欧

  • 主要君主：亚历山大（就一个）

  • 制度：沿袭波斯帝国

    • 国王、君主专制
    • 君权神授
    • 地方行省制
    • 马其顿人 & 希腊人 -> 担任主要职务

  • 文化：推广希腊文化

罗马

  • 罗马共和国 -> 东西罗马帝国
  • 兴起：
    原来是意大利中部的一个城邦
    公民组成的强大的军队 -> 征服整个地中海周边地区
    -> 地中海成为罗马的内海
  • 奴隶制
  • 宗教：1世界，基督教产生于巴勒斯坦，4世纪末成为罗马帝国国教

  • 衰落及灭亡

    • 3世纪后，官僚体系、职业军队 -> 人民负担不断加重 & 边境压力加重
    • 4世纪末，分裂成东西两部分
    • 5世纪后期，内外矛盾，西罗马帝国灭亡
    • 东罗马帝国一直到15世纪
