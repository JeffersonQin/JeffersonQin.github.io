第四章 - 剖析物质变化中的能量变化

物质在溶解过程中有能量变化

物质变化中的能量变化：

  1. 能量守恒定律：能量在各种形式之间的转换和各个物体之间的转移过程中，各种形式的能量综合保持不变。
  2. 物理变化：意义：$1 \text {mol}$液态水变成气体，吸收$44\text { kJ}$的热量。
  3. 化学变化：意义：$1 \text { mol}$的固态$\ce {C}$和$1 \text { mol}$的氧气反应，生成一摩尔的二氧化碳气体并放出393.5千焦的热量。
    要注意量、状态

溶解过程和溶解热现象

  • 溶液温度升高：$\ce {NaOH (s)、浓硫酸}$
  • 溶液温度降低：$\ce {NH4NO3 (s)、NH4Cl (s)}$
  • 溶液温度变化不大：$\ce {NaCl (s)}$

解释：

比如：$\ce {Cu2+}$和水发生水合过程，生成水合铜离子，显蓝色

  • $扩散过程中吸收的热量 > 水合过程中放出的热量 \rightarrow 溶液温度下降$
  • $扩散过程中吸收的热量 < 水合过程中放出的热量 \rightarrow 溶液温度升高$
生成溶液的过程既不是物理变化，也不是化学变化，是物理化学变化

溶解和结晶

  • 溶解：固体溶质扩散到溶剂中去的过程
  • 结晶：固体溶质从溶液中析出的过程
溶解过程和结晶过程同时存在，是一组相反的过程
  • $v_{溶解} > v_{结晶}$，固体溶解（减少），不饱和溶液
  • $v_{溶解} < v_{结晶}$，固体析出（增加），饱和溶液
  • $v_{溶解} = v_{结晶}$，溶解平衡，饱和溶液
溶解平衡：与外界条件（温度、溶剂量等）有关
改变外界条件（升温、加水等），原来的平衡被破坏，在新的条件下达到新的平衡 -- 平衡发生移动。

结晶水合物

  • 晶体：自发形成的具有规则的几何形状的固体
  • 结晶水合物：含有结晶水的固体
  • 风化：室温下干燥空气中失去部分或全部结晶水的现象
  • 潮解：吸收空气中的水蒸气在固体表面形成溶液的现象
    • 例如氢氧化钠作为干燥剂

4/23 化学课 - 可逆反应与平衡状态

可逆反应

正反应：

逆反应：

  1. 定义：在同一条件下，既能向正反应方向进行又能向逆反应方向进行的反应
  2. 特点
    • 在同一条件下进行
    • 正逆反应同时存在
    • 反应物生成物共存
  3. 表示方式：可逆反应化学方程式中常用 $\ce {<=>}$

溶解平衡

在一定条件下,当溶质溶解的速率等于结晶的速率,固体溶质不再减少,也不再增加,溶液处于饱和状态,称之为达到了溶解平衡状态,简称溶解平衡。

化学平衡状态

  • 在一定条件下的可逆反应中,正反应速率和逆反应速率相等,反应混合物中各组分的浓度保持不变的状态,称之为化学平衡状态,简称化学平衡。
  • 化学平衡状态是可逆反应在一定条件下所能达到的最大限度。

化学平衡的特征

  1. 研究对象：可逆反应
  2. 实质：$v_正 = v_逆$
  3. 达到化学平衡时，正逆反应还在进行：$v_正 = v_逆 ≠ 0$
  4. 现象：达到化学平衡时反应混合物中各物质的浓度不再发生变化
  5. 若改变浓度、温度、压强等反应条件，化学平衡会被破坏

4/23 地理课 - 高新技术产业区位

高新技术产业

  1. 基本概念：高新技术产业是以微电子技术与计算机技术、信息通讯技术、生物技术、新材料技术、激光技术、航天技术为支撑的产业。
  2. 特点：
    1. 科研人员占职工总人数的比例高
    2. 研发费用占销售收入的比例高
    3. 产品更新换代比较快
    4. 产品附加值高
  3. 意义：
    1. 节约劳动力，提高生产效率
    2. 提高资源、能源的利用率
    3. 拓展新领域、创造新产业
  4. 产品特点：轻、小、精、贵。
  5. 区位条件：
    1. 原料、能源、劳动力价格和距离市场远近等因素的影响在减弱或相对下降。
    2. 技术集聚程度、人才、信息、环境、交通、通讯等因素的影响力在增强

案例分析：美国硅谷

  1. 位置：硅谷位于美国西部、圣弗朗西斯科湾区南部, 是高科技事业云集的美国加州圣克拉拉谷的别称。
  2. 环境：美国硅谷环境优美；有着冬暖夏凉的地中海气候。这里三面环水,深受海洋的影响并受太平洋加利福尼亚寒流影响夏季多雾。
  3. 技术人才：斯坦福大学在美国硅谷的形成中起了关键作用,这里的许多大学是硅谷微电子产业的孵化器,也为硅谷输送大量的技术人才。
  4. 交通：邻近旧金山航空港; 高速公路贯穿全境。
  5. 市场：
    • 美国政府一直维持着对“硅谷“半导体产品稳定的订货量。
    • 产品面向世界市场
  6. 文化：崇尚创新、鼓励冒险、宽容失败、诚心协作。

小结

  • 核心内容是农业和工业区位条件及变化
  • 通过对典型案例分析的方法,加深对农业和工业区位以及农业地域类型和工业区的认识;以某一区域农业和工业的区位变化为例,了解社会经济发展对区位变化的影响。
  • 树立因地制宜、因时制宜发展农业的观念;增强对农民、农业、农村问题的关注;感悟工业发屐对社会经济发展的影响;体会“创新驱动、转型发展”对区域产业发展的价值。

几何级数的敛散性

根据等比数列前$n$项求和公式，若$|q| ≠ 1$，则部分和为

  • 当$|q| < 1$时： 级数收敛
  • 当$q = 1$时： 级数发散
  • 当$q = -1$时： 级数发散
  • 当$|q| > 1$时： 级数发散

综上所述：

  • $|q| < 1$时，几何级数$\displaystyle \sum_{n = 0} ^ \infty aq ^ n$收敛，其和为$\displaystyle\frac a {1 - q}$。
  • $|q| ≥ 1$时，几何级数$\displaystyle \sum_{n = 0} ^ \infty aq ^ n$发散。

交错级数审敛法 - The Alernating Series Test for Convergence

莱布尼兹判别法：如果交错级数满足条件：

  1. $u_n ≥ u_{n + 1}, (n = 1, 2, 3, \cdots)$
  2. $\lim_{n \rightarrow \infty} u_n = 0$

则交错级数收敛，且其和$S ≤ u_1$，用它前$n$项的部分和$S_n$作为级数的和$S$的近似值时，误差$|S_n - S| ≤ u_{n + 1}$

比值判别法 - Ratio Test for Convergence

设有正项级数$\displaystyle\sum_{n = 1} ^ \infty u_n$，如果$\displaystyle\lim _{n \rightarrow \infty} \frac {u_{n + 1}}{u_n} = l$，则

  • 当$l < 1$时，级数收敛
  • 当$l > 1$（或$l \rightarrow \infty$）时，级数发散
  • 当$l = 1$时，级数可能收敛也可能发散

催化剂对化学反应速率影响的实验研究

  1. 研究对象：$\ce {2H2O2 ->[MnO2]2H2O + O2 ^}$
  2. 实验：在双氧水中加入二氧化锰
  3. 现象：没有加入$\ce {MnO2}$粉末的双氧水试管中只有少量气泡出现,不能使带火星的木条复燃;加入$\ce{MnO2}$粉末后试管内产生大量气泡,带火星的木条复燃
  4. 结论：催化剂可以有效地提高化学反应速率

结论：

  • 加入催化剂改变了反应的途径,降低了反应所需要的能量,从而加快化学反应速率。
  • 加入催化剂不会改变该化学反应的热效应。

压强对化学反应速率的影响

结论：对于有气态物质参加的反应,在其它条件不变时增大压强,化学反应速率加快;减小压强,化学反应速率减慢。

如果参加反应的物质全部是固体、液体或溶液时,改变压强对它们的反应速率无影响。
改变压强一般指改变体积。如果是充入其他气体，对气体浓度没有影响，所以不会有什么影响。

4/21 地理课 - 世界主要工业区

北美工业区

  • 主要分布区：美国东北部和加拿大东南部
  • 区位条件：
    1. 丰富的煤、铁及水资源
    2. 交通便利
    3. 城市人口众多，市场广阔
    4. 农产品供应充足
  • 资料:底特律始建于1701年,1830年后工业稳步成长,1903年世界第一家汽车公司建立,此后迅速发展成为世界著名的“汽车城市”。20世纪70年代受到石油危机重创,加上日本、德国等国汽车生产的冲击,开始走向衰落20世纪90年代以后努力发展第三产业,但未能扭转经济衰退趋势。2007年房地产市场崩溃,2013年2月19日底特律市宣布已经陷入财政危机.
  • 衰退原因：
    1. 汽车产业为主,工业结构单一;
    2. 石油危机;
    3. 日本等国家汽车工业的冲击;
    4. 高新技术产业的冲击;
    5. 环境污染严重;
    6. 房地产崩溃。
  • 扩展方向：国际市场竞争激烈,许多工业部门的优势逐渐降低。工业地域由美国东北部地区向南部和西部扩展。
  • 向南部和西部迁移的原因：
    1. 东北地区的工业衰退,失业率升高;
    2. 西部和南部建立了许多新兴的工业部门
    3. 南部和西部环境优美,环境污染少。

西欧工业区

  • 主要分布区：包括英国、法国、德国、荷兰、比例式、意大利北部等主要工业区
  • 区位条件：
    1. 煤、铁资源丰富;
    2. 交通便利;
    3. 劳动力素质高;
    4. 经济基础雄厚。
  • 主要特点：是现代工业的发源地,核能、航空等技术密集型工业最为发达,钢铁、机床、船舶、电力等产品产量居世界前列。
  • 扩展方向：工业地域逐渐从内陆向沿海地区发展。

东欧工业区

  • 主要分布区：包括俄罗斯、乌克兰、捷克、波兰等的主要工业区。
  • 区位条件：
    1. 丰富的资源
    2. 基础工业实力雄厚
  • 主要特点：属资源密集型重工业地区,工业多布局在靠近原料和燃料产地的内陆。
  • 发展方向：电子、宇航等新兴工业发展较快,工业地域逐渐由乌拉尔山以西的工业区向以东的地区扩展。
  • 向东发展的原因：乌拉尔山以东地区的资源储量占全国总量的80%以上,但开采利用较少。随着欧洲部分自然资源的枯竭和工业区的“过饱和”,为了充分利用东部地区的资源,工业重心有向东转移的趋势

日本太平洋沿岸工业区

  • 主要分布区：分布在日本太平洋沿岸地区
  • 高度集中在太平洋沿岸的原因
    1. 日本国内发展工业所需的资源贫乏,绝大多数依靠海外进口;
    2. 日本国内市场有限,产品需要大量销往海外;
    3. 日本是岛国,海岸线漫长,多优良港湾;
    4. 资源来源主要面向太平洋一侧。
  • 区位条件：
    1. 劳动力素质高
    2. 科技力量强
    3. 海运便利
  • 主要特点：加工贸易型,有世界上著名的临海工业带。
  • 发展方向：技术密集型产业发展迅速,工业地域分布有明显的分散趋势。

亚洲东部沿海工业区

  • 主要分布区：北起韩国东南沿海和中国东部沿海,向南延伸到东南亚一些国家。
  • 区位条件：
    1. 劳动力资源丰富且价格低;
    2. 基础设施逐步完善;
    3. 土地价格便宜;
    4. 国家优惠政策支持;
    5. 市场广阔。
  • 发展方向：由劳动密集型转向技术密集型
  • 转型原因：随着区域经济的发展,劳动力成本增加,劳动密集型工业成本优势丧失,这里的劳动密集型产业转向更具成本优势的区域;劳动力素质的提高使工业向技术密集型转型成为可能。
  • 中国东部沿海工业基地
    • 辽中南：著名的重工业基地
      • 优势区位条件:
        1. 丰富的矿产资源;
        2. 便利的海陆交通;
        3. 雄厚的基础;
        4. 振兴东北政策。
      • 特点:重工业基地。
      • 存在问题: 能源枯竭、污染严重、产品单一、技术不足、水源不足。
    • 京津唐：北方最大的综合性工业基地
      • 优势区位条件
        1. 丰富矿产资源;
        2. 交通便利;
        3. 农业基础好;
        4. 科研机构和高校依托;
        5. 京津冀协同发展政策。
      • 特点:综合型工业基地。
      • 存在问题:水源不足、环境污染严重。
    • 沪宁杭：全国最大的综合性工业基地
      • 优势区位条件
        1. 交通便利;
        2. 经济活跃;
        3. 工农业基础雄厚;
        4. 技术力量雄厚;
        5. 长三角一体化政策
      • 特点:综合型工业基地。
      • 存在问题:能源、矿产资源缺乏。
    • 珠江三角洲：轻工业为主的综合性工业基地
      • 优势区位条件
        1. 邻近港澳,靠近东南亚;
        2. 劳动力资源丰富;
        3. 交通便利,基础设施完善;
        4. 开放意识,政策支持;
        5. 投资环境好;
        6. 水资源丰富。
      • 特点:轻工业为主的综合型工业基地。
      • 存在问题:能源、矿产资源不足。
