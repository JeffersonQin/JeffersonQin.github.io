4/7 化学课

固氮

  1. 大气固氮
    $$\ce {N2 + O2 ->[放电] 2NO}$$
  2. 生物固氮
    $$\ce {N2 ->[固氮酶][H^+] 氨基酸}$$
  3. 人工合成固氮
    $$\ce {N2 + 3H2 <=>[高温、高压][催化剂] 2NH3}$$

氮气与氮氧化物

  1. $\ce {NO}$无色气体
  2. $\ce {NO2}$红棕色气体
  3. $\ce {N2}$无色无味气体

$$\ce {N2 + 3Mg ->[点燃] Mg3N2}$$
$$\ce {N2 + 3H2 <=>[高温高压][催化剂]2NH3}$$
$$\ce {N2 + O2 ->[放电] 2NO}$$
$$\ce {2NO + O2 -> 2NO2}$$
$$\ce {3NO2 + H2O -> 2HNO3 + NO}$$

氨气

  1. 水溶液呈碱性
    $$\ce {NH3 + H2O <=> NH3H2O <=> NH4+ + OH-}$$
    $$\ce {3NH3    H2O + Fe^3+ -> Fe(OH)3 v + 3NH4+}$$
  2. 与酸反应生成铵盐
    $$\ce {NH3 + H+ -> NH4+}$$
    $$\ce {NH3 + HCl -> NH4Cl}$$
    $$\ce {2NH3 + H2SO4 -> (NH4)2SO4}$$
  3. 还原性
    $$\ce {4NH3 + 5O2 ->[催化剂][加热] 4NO + 6H2O + Q}$$
    $$\ce {4NH3 + 3O2 ->[点燃] 2N2 + 6H2O}$$

铵盐

  1. 不稳定性
    $$\ce {NH4Cl ->[\Delta] NH3 ^ + HCl ^}$$
    $$\ce {NH4HCO3 ->[\Delta] NH3 ^ + CO2 ^ + H2O}$$
    $$\ce {(NH4)2CO3 ->[\Delta] 2NH3 ^ + CO2 ^ + H2O}$$
    $$\ce {(NH4)SO4 ->[\Delta] NH3 ^ + NH4HSO4}$$
    $$\ce {2NH4NO3 ->[\geq 300^\circ C][或撞击] 2N2 ^ + O2 ^ + 4H2O}$$
    碳酸氢铵和氯化铵的分解现象不一样，氯化铵分解生成的氨气和氯化氢可以再化合生成氯化铵。
  2. 与碱反应
    $$\ce {NH4+ + OH- ->[\Delta] NH3 ^ + H2O}$$
    $$\ce {NH4Cl + NaOH ->[\Delta] NaCl + NH3 ^ + H2O}$$
    $$\ce {NH4NO3 + NaOH ->[\Delta] NaNO3 + NH3 ^ + H2O}$$
    $$\ce {(NH4)2SO4 + 2NaOH ->[\Delta] Na2SO4 + 2NH3 ^ + 2H2O}$$
    因为氨气极易溶于水，所以需要加热才能放出气体
  3. 应用：
    • 制氨气
      $$\ce {(NH4)2SO4 + Ca(OH)2 ->[\Delta] CaSO4 + 2NH3 ^ + 2H2O}$$
      $$\ce {s + s / \Delta}$$
      氨气不可以用无水$\ce {CaCl2}$干燥，用碱石灰，氢氧化钠固体干燥。
    • 检验铵盐：取样、加氢氧化钠溶液、加热、产生的气体使湿润的红色石蕊试纸变蓝——有$\ce {NH4+}$
    • 铵态氮肥的储存——阴凉处
    • 氮肥不能与碱性物质混用

4/3 地理课 - 农业区位条件

  1. 基本概念：农业区位条件是指影响农业生产布局的各种自然因素和社会经济因素。

$$
农业区位因素
\begin{cases}
自然因素 \begin{cases}
气候 \ 地形 \ 土壤 \ 水源 \ \cdots
\end{cases} \ 社会经济因素 \begin{cases}
市场 \ 交通 \ 劳动力 \ 政策 \ 技术 \ \cdots
\end{cases}
\end{cases}
$$

影响农业生产布局的自然因素

  1. 气候因素
    1. 光照：光照长短和强度影响作物的品质和产量。
    2. 热量：热量影响农作物的品种、生长期和熟制。
    3. 水分：水分音响主要农作物的品种和农业生产方式。
  2. 地形因素：不同的地形区，示意发展不同生产方式的农业。
    1. $坡度 < 15 ^\circ$：农田
    2. $15 ^\circ < 坡度 < 25 ^\circ$：林地为主，部分为梯田
    3. $25 ^\circ < 坡度 < 35 ^\circ$：主要为林牧用地
    4. $坡度 > 35 ^\circ$：封山育林区
  3. 土壤因素：土壤是作物生长的物质基础，不同的土壤类型适宜生长不同的作物。
  4. 水源因素：水源是农业生产的必备条件。
    干旱地区：旱作农业或灌溉农业。主要分布在河流沿岸和山麓绿洲地区。因为这里深居内陆，远离海洋，气候干旱，只有灌溉水源充足的地方才能发展种植业。

影响农业生产布局的社会经济因素

  1. 市场因素：市场的需求应到这农业生产的类型、规模和布局。
    随着城市化的发展，城市人口增多，人们对生活水平的提高，人们对瓜果、蔬菜、乳、肉、禽、蛋的需求量不断增加。
  2. 交通因素：交通运输是农业区位选择的重要条件，交通运输的发展扩大了农业生产的市场范围。
  3. 劳动力因素：劳动力数量、素质影响农业区位选择。
  4. 政策因素：国家政策也会对农业区位产生深远的影响。
    材料：为确保18亿亩国家耕地红线，我国制定了基本农业天保护条例，不准占用基本农田发展林果业、不准在基本农田内挖塘养鱼和进行畜禽养殖。确保国家粮食安全底线。
  5. 技术因素：农业科学技术影响着农业生产方式，依靠技术可以培育优良品种，提高弄哦单产。

案例分析：美国乳蓄带的区位条件

  1. 主要分布在美国的东北部和五大湖的滨湖地区。
  2. 自然因素
    1. 气候：温带大陆性气候，气候温凉潮湿。
    2. 地形：地形平坦，受众多河湖影响，未能集中连片。
    3. 土壤：图层薄，土地贫瘠。
    4. 水源：河湖众多，水源充足。
    5. 结论：这里气候温凉潮湿；地形平坦；土壤贫瘠；水源充足。不利于谷物生长，有利于多汁牧草生长，适合发展乳畜业。
  3. 社会经济因素
    1. 市场：人口密集，城市集中，经济发达，消费市场广阔。
    2. 技术：欧洲移民带来饲养奶牛和种植牧草的先进经验和技术。
    3. 交通：交通便利，冷藏保鲜运输技术发达。
    4. 工业：为乳畜业生产提供机械。
    5. 结论：这里消费市场广阔；拥有先进经验和技术；交通便利；工业基础好；有利于乳畜业形成专门化生产。

学会评价某一区域的农业区位条件

  • 评价：
    • 有利：人口密集，经济发达，市场需求量大
    • 不利：缺少优质天然草场。
  • 建议：加大科技投入，培育优良的畜种，提高乳蓄产品质量；研制开发青饲料、精饲料，缓解天然草场不足的问题。推进乳蓄产品的加工，延长产业链，增加附加值；利用遍历的交通、通讯网络拓展消费市场；利用区位优势，发展特色旅游等。

铵盐

物理性质

  1. 都是晶体
  2. 一般能溶于水

结构

  1. 离子化合物
  2. 能电离出铵离子

化学性质

  1. 与$\ce {OH-}$反应
    $$\ce {NH4+ + OH- ->[\Delta] NH3 ^ + H2O}$$
    实验室制备氨气：

    • 气体发生装置：固固加热
    • 药品：氢氧化钙固体 + 铵盐
      • 注：氢氧化钠和氢氧化钾腐蚀玻璃太厉害了，所以选用氢氧化钙
    • 向下排空气法收集（氨气密度比空气小）
    • 湿润的红色石蕊试纸检验
      • $$\ce {NH3 + H2O <=> NH3*H2O <=> NH4+ + OH-}$$

  2. 受热分解
    $$\ce {NH4HCO3 ->[\Delta] NH3 ^ + H2O + CO2 ^}$$
    $$\ce {NH4Cl ->[\Delta] NH3 ^ + HCl ^}$$
    $$\ce {NH3 + HCl -> NH4Cl}$$

    受热分解 – 低温、密封、深施盖土
    与碱反应生成氨气 – 不能与碱性物质混合使用

  3. 尿素：$\ce {CO(NH2)2}$

4/1 地理课

城市问题

  1. 基本概念：随着城市化进程的推进，城市人口迅速增长，城市规模不断扩大，在创造由于物质生活条件的同时，也带来了许多问题，这些问题统称为城市问题
  • 环境问题：空气、水、垃圾、噪声等污染
  • 交通问题：车流量过大，道路不看重负，交通堵塞严重
  • 居住问题：住宅密度过高，居住面积狭小，舒适性差
  • 社会问题：就业压力大、社会治安恶化

产生原因

  • 城市人口过快增长和城市的无序扩张，规模过大、过于集中，城市基础设施不敷使用。
  • 对急剧膨胀真被不足，城市规划跟不上发展，内部功能分区不合理，城市管理和视屏素质水平有待提升。
  • 片面追求理论和速度，忽视工作、生活等环境建设。
  • 交通堵塞，能耗和排放巨大，城市环境不断恶化。

解决措施

  1. 基本概念：城市问题是城市哦发展到一定阶段的必然产物，各种问题相互影响，相互制约。解决城市问题是一项系统工程。宏观上，要发展经济、政策调控、合理规划，完善法律；微观上，要针对具体城市问题有的放矢，采取可续合理可行的措施

宏观层面

  • 进行科学合力的城市规划，开发新区，建立卫星城市，分散城市职能。
  • 完善现代城市管理制度，优化大城市治理体系。

微观层面

  • 环境问题：保护和智力环境、减少污染排放、加强绿化建设
  • 交通问题：积极发展共欧诺个交通、大力发展节能环保“绿色交通”，如轨道交通和大运量公交等，合理规划城市交通设施。
  • 居住问题：改善居住条件、推广廉价租房等。
  • 社会问题：提高就业率、完善社会保障和治理体系。

3/31 化学课

固氮

  1. 定义：把大气中游离态的氮转化为氮的化合物的过程
  2. 途经
    1. 大气固氮（打雷，氧气和氮气反应生成一氧化氮）
      $$\ce {O2 + N2 ->[放电] 2NO}$$
      $$\ce {2NO + O2 -> 2NO2（红棕色）}$$
      $$\ce {3NO2 + H2O -> 2HNO3 + NO}$$
      $$\ce {NH3 + HNO3 -> NH4NO3}$$
      重点：这里会发生一系列反应，是考察重点。
    2. 生物固氮（根瘤菌）
      $$\ce {N2 ->[固氮酶][H^+] 氨基酸}$$
    3. $$\ce {N2 + 3H2 <=>[高温、高压][催化剂] 2NH3}$$

氨气

  1. 分子结构
  2. 物理性质
    • 无色、具有刺激性气味的气体
    • 密度比空气小
    • 易液化
    • 极易溶于水：常温下，1体积水溶解700体积氨气（防止倒吸）
  3. 化学性质
    1. 水溶液呈碱性
      $$\ce {NH3 + H2O <=> NH3*H2O <=> NH4+ + OH-}$$
      一水合氨：一元弱碱，浓氨气可以挥发出氨气。
      使用浓氨水和生石灰制备氨气：
      $$\ce {CaO(s) + H2O(l) -> Ca(OH)2 + Q} \Rightarrow T \uparrow$$
    2. 与酸反应（生成铵盐）
      $$\ce {NH3 + H+ -> NH4+}$$
      $$\ce {NH3 + HCl -> NH4Cl}$$
      浓盐酸和浓氨水 -> 白烟（浓硫酸不挥发，看不到）
      $$\ce {2NH3 + H2SO4 -> (NH4)2SO4}$$
      注意和碳酸反应是可能生成碳酸铵和碳酸氢铵，取决于过量和不过量。
      $$\ce {NH3 + H2O + CO2 -> NH4+ + HCO3-}$$
      $$\ce {3NH3*H2O + Fe^3+ -> Fe(OH)3 v + 3NH4+}$$
    3. 还原性（催化氧化）
      $$\ce {4NH3 + 5O2 ->[催化剂][加热] 4NO + 6H2O }$$
      在氧气中安静地燃烧
      $$\ce {4NH3 + 3O2 ->[点燃] 2N2 + 6H2O}$$
      注意：这里氮气和氧气在不同的条件下会发生不同的反应，需要格外注意。

3/31 地理课

城市化的主要阶段

  1. 城市化初期：人口和资本向中心城区聚集
    • 城市人口迅速增长
    • 城市用地规模扩大
  2. 郊区化时期：人口和资本从城区向郊区迁移
    • 中心城区人口迁往郊区
