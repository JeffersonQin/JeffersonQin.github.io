5/15 世界文化圈

基本概念

文化圈又称文化区,指具有相似的**文化现象和生存方式**的地理区域
  • 表现: 语言文字、生活习惯、宗教信仰、思想观念等比较类似, 并有共同的渊源和紧密的联系。
  • 划分: 文化圈的划分以文化源地为中心, 以民族为单位进行组合。

世界文化圈

  1. 西欧文化圈
    • 分布地区：西欧、北美和澳大利亚等地区
    • 特点：西欧文化发源地，深受英国影响
    • 代表文化事物和现象：
      • 乳畜生产文化,普遍使用刀叉
      • 信奉基督教新教、罗马天主教
      • 不同风格的西方古典建筑
      • 各种基督教节日庆典
  2. 东欧文化圈
    • 分布地区：以斯拉夫民族为主的东欧地区
    • 特点：深受俄罗斯影响
    • 代表文化事物和现象：
      • 信奉东正教
      • 圆形穹顶的东正教堂
      • 俄罗斯精湛的芭蕾舞技术
  3. 东亚文化圈
    • 分布地区：中国、日本、朝鲜半岛、蒙古
    • 特点：深受中国古代文化,特别是儒家思想和汉字影响
    • 代表文化事物和现象：
      • 稻米生产文化,普遍使用筷子
      • 中国旗袍、日本和服、朝鲜高腰长裙、蒙古袍等
  4. 南亚文化圈
    • 分布地区：印度、巴基斯坦、尼泊尔、孟加拉国、斯里兰卡等
    • 特点：
      • 深受古印度文明影响
      • 民族、语言、宗教都比较复杂
    • 代表文化事物和现象：
      • 印度教、锡克教、佛教、伊斯兰教等多种宗教
      • 泰姬陵、印度舞蹈、纱丽服饰
  5. 东南亚文化圈
    • 分布地区：中南半岛、马来群岛
    • 特点：受中国、印度及欧美影响,文化丰富多样,体现多元特色
    • 代表文化事物和现象：
      • 主要信奉佛教和伊斯兰教(菲律宾以天主教为国教)
      • 吴哥窟(柬埔寨)、婆罗浮屠(印度尼西亚)
  6. 伊斯兰文化圈
    • 分布地区：中亚、西亚和北非地区
    • 特点：阿拉伯民族为主、典型的伊斯兰宗教文化地区
    • 代表文化事物和现象：阿拉伯长袍、清真寺、穆斯林
  7. 非洲文化圈
    • 分布地区：撒哈拉以南的非洲地区
    • 特点：
      • 以黑色人种为主,民族、宗教复杂
      • 崇尚自然,多信奉原始宗教
    • 代表文化事物和现象：
      • 原始的图腾崇拜
      • 神秘的宗教仪式
      • 刚劲粗犷的鼓声和舞蹈
  8. 拉丁美洲文化圈
    • 分布地区：墨西哥及其以南的美洲地区
    • 特点：外来文化与本地文化相互融合
    • 代表文化事物和现象
      • 原始的图腾崇拜
      • 神秘的宗教仪式
      • 刚劲粗犷的鼓声和舞蹈
  9. 太平洋文化圈
    • 分布地区：太平洋中的岛屿,包括波利尼西亚、密克罗尼西亚、美拉尼西亚三大岛群
    • 特点：居民原崇拜多种神灵,相信巫术,后多改信基督教
    • 代表文化事物和现象：
      • 擅长航海
      • 土风歌舞

工业制硫酸

硫酸的用途与我国硫酸产量

第一阶段：二氧化硫的制备

$$\ce {S + O2 ->[点燃] SO2}$$
或者
$$\ce {4FeS2 + 11O2 ->[煅烧] 2Fe2O3 + 8SO2}$$

  • 把硫铁矿矿石粉碎成细小的颗粒,然后把它放在沸腾炉中进行煅烧。
  • 将矿石粉碎成很细的颗粒,增大了矿石颗粒的表面积,有利于矿石与空气充分接触,加快煅烧速率。
  • 矿石颗粒在炉内一定空间里剧烈翻腾,与空气接触充分,煅烧速率快,煅烧充分

第二阶段：二氧化硫的催化氧化

$$\ce {2SO2 + O2 <=>[催化剂][\Delta] 2SO3}$$

第三阶段：三氧化硫的吸收

$$\ce {SO3 + H2O -> H2SO4}$$

原料选择

  1. 硫铁矿制硫酸
  2. 硫磺制硫酸
  3. 冶炼烟气制硫酸
  4. 其他

二氧化硫的催化氧化

  • 生产设备：接触室
  • 基本过程:精制后的炉气补充空气后送入装有五氧化二矾催化剂的接触室,氧化生成$\ce {SO3}$。

三氧化硫的吸收和硫酸生成

  • 用98.3%的硫酸吸收SO3,不易形成酸雾,吸收速率快。
  • SO3从塔的下部通入,98.3%的硫酸从塔顶喷下,两者逆流接触,吸收充分。

5/13 地理课 - 民族文化、地方文化及文化整合

民族文化和地方文化

  • 民族
    • 民族是指不同地域的人,由于经济生活、语言文字、风俗习惯和历史发展的不同而形成的社会共同体。
    • 世界上有2000多个民族,人口最多的是汉族,人数最少的民族只有几十个人。
    • 一般,各个民族都有自己的民族语言,但也有几个民族使用同一种语言,或个别的民族使用两、三种不同语言。
    • 民族是历史文化的产物,在创造文化的过程中形成民族文化的人文特色。
    • 民族无高低贵贱之分,只有生产力发展水平、文化先进与落后的差异。这种差异是在自然、社会环境中后天形成的,是一种历史现象,而不是永恒的。

不同民族的人文特色

  • 中华民族 – 勤劳刻苦
  • 法兰西民族 -— 热情浪漫
  • 德意志民族 —- 严谨善思
  • 英吉利民族 – 矜持幽默
  • 美利坚民族 – 冒险求新

民族与种族

  • 民族:不同地域的人,由于经济生活、语言文字、风俗习惯和历史发展的不同而形成的社会共同体。
    • (属于社会历史范畴)
  • 种族:体质形态上具有某些共同遗传特征的人群,是人类对自然环境长期适应的结果。
    • (属于自然范畴)
  • 世界各地不同人群形成的地域文化具有地方性、民族性、民间性和传统性,是经过长期的交流和选择而传承下来的。
  • 民族文化和地方文化具有鲜明特征,只有民族的才是世界的。
  • 世界文化多姿多彩,是各民族、各地方优秀文化的集成。

保护民族文化和地方文化的意义

  • 就是保护人类文化,有利于人类文化的发展和进步。
  • 是研究古代文明的重要依据,有利于对古代文明的研究
  • 是对少数民族、土著居民的尊重,有利于民族团结和社会稳定。
  • 是重要的旅游资源,有利于促进当地社会、文化、经济的发展。
  • 是文化多样性的体现，一旦消失就难以再现，是世界文化的损失
  • ……

文化整合

  1. 文化整合是指各种地方文化、民族文化通过接触、交流、传播、融合和吸收,形成新的文化的过程。
  2. 文化整合不是一种文化取代另一种文化,而是各种文化的协调、发展和提升。
新文化的进步是融合吸收其他文化,摒弃落后过时文化,吐故纳新的过程。

影响文化整合的因素

  • 环境因素————相同、相近的自然环境有助于文化整合。
  • 社会因素————文化整合与政治、经济因素有密切的关系。
  • 人口、民族迁移因素————造成不同文化的碰撞、交流和整合。
  • 文化自身因素————优秀、合理、符合时代要求的先进文化,在文化整合中发挥巨大的作用。
  • 时间因素————需要长时间的选择和吸收,逐渐融合。

文化整合的社会意义

通过文化整合,提高内部凝聚力,实现民族团结,促进各民族文化和地方文化的交流融合,推动各民族文化和国家乃至世界整体文化的繁荣和发展。

5/13 物理课 - 重力势能

  1. 物体由于被举高而具有的能量叫做重力势能
  2. 物体的重力势能与物体的质量和物体所处的高度有关
  3. 重力势能的大小
    $$E_p = mgh$$
  4. 重力势能的单位：焦耳 (J)
  5. $h$为物体距离零势能面的距离
  6. 零势能面通常选地面
  7. 重力势能具有相对性，是一个相对量
  8. 重力势能是标量，但是有正负
  9. 重力势能的变化与零势能面的选取无关
  10. 弹性势能
    $$E_p = \frac 1 2 k \Delta x ^ 2$$

勒夏特列原理与应用

勒夏特列原理

勒夏特列原理(又称平衡移动原理): 如果改变影响平衡的个条件(如浓度、温度或压强等),平衡就向能够减弱这种改变的方向移动。

注：是减弱，不是抵消
  • 条件改变,平衡移动只是减弱外界条件带来的变化而不能完全抵消外界条件的变化。
  • 从平衡移动的结果看,外界因素变化对平衡的影响占主导地位
  • 勒夏特列原理既指明了条件改变,化学平衡移动的方向,也指明了化学平衡移动的结果。

勒夏特列原理的适用范围

  • 适用于已达平衡的体系,不适用于未达平衡的体系。
  • 适用于浓度、温度、压强等因素改变导致化学平衡发生移动的问题
  • 不能用来判断新平衡建立所需要的时间
  • 不仅适用于化学平衡，也适用于溶解平衡等其他动态平衡

5/12 地理课 - 文化与环境

基本概念

文化是指人类在社会实践过程中所获得的物质的、精神的生产能力和创造的物质、精神财富的总和

文化的两种形态

  • 物质文化：为了满足人类生存和发展需要所创造的物质产品及其所表现的文化。包括饮食、服饰、建筑、交通、生产工具以及乡村、城市等。
  • 非物质文化
    • 精神文化:是文化的核心,包括价值观、道徳伦理、民族性格、科学理论、文学艺术、宗教哲学等。
    • 制度文化:政治、经济、法律、军事、教育、婚姻等一系列制度以及组织机构。

环境对文化的影响

  • 从本质上说, 文化是人与环境相互作用的产物。环境包括自然地理环境和人文社会环境。
  • 环境与文化之间相互影响、相互作用。环境对文化起着非常重要的作用,而文化一经形成又会对环境产生影响。

环境对物质文化的影响

  • 直接制约和影响物质文化的发展,使其对环境具有强烈依赖性
  • 环境、资源等自然条件决定人工制品的材料、样式和风格
  • 环境影响产业布局
  • 环境影响社会发展水平

环境对非物质文化的影响

  • 非物质文化被打上深深的环境焙印
  • 影响原始宗教和信仰的形成和发展
  • 影响人们的生产生活方式、思想观念、风俗习惯
  • 影响文学艺术
  • 影响语言
  • …

例子：民居屋顶的差异

分属干旱地区和湿润地区

  • 干旱地区降水较少,平顶既可节省建筑材料,还可兼作晾晒农作物的场所。
  • 湿润地区年降水量大,夏季炎热,斜坡式的屋顶既利于排水,又利于通风散热。

例子：不同的传统民居

  • 吊脚楼是西南地区少数民族传统民居,一般为竹木结构,高悬地面既通风干燥,又能防毒蛇、野兽,楼板下还可放杂物。
  • 窑洞是中国北方黄土高原上特有的民居形式,直立性很强的黄土为窑洞提供了很好的发展前提。窑洞防火,冬暖夏凉,经济实用。
  • 粘土地区 - 砖瓦结构
  • 山区 - 石木结构
  • 黄土地区 - 陕北窑洞
  • 湿热地区 - 竹木建筑
  • 水乡地区 - 砖木结构

例子：蒙古包和毡房

内陆草原地区:蒙古包、哈萨克毡房,拆卸方便,坚固耐用…

例子：南米北面

与我国南北方农业生产结构不同有关：

  • 南方地区夏季炎热多雨,水分、热量充足,耕地以水田为主,因地制宜种植生长习性喜温喜湿的水稻。
  • 北方地区降水较少,气温较低,耕地多为旱地,适合喜凉耐旱的小麦生长。

例子：南船北马

  • 南方地区气候湿润,降水丰富,地表河网密布,地形较复杂,运输以船舶为主
  • 北方地区气候较为干燥,地势比较平坦,道路相对较宽,多陆踣运输;且草场广布,畜牧业发达;因此马匹以其耐力好、速度快而被驯化为交通工具。

例子：南繁北齐

  • 南繁北齐,我国语言的地理差异表现为南繁北齐,即南方语言繁杂,北方语言比较单一。南方即使在同一方言区内,如闽方言区内,还分福州话、厦门话、莆田话、闽西话等,彼此差异极大,甚至听不懂。
  • 南繁北齐的形成与地理环境存在一定关系。我国北方多地势平坦的高原和平原,交通联系方便,老百姓交往、交流的机会多,彼此融合,因而语言差异不大。而南方多丘陵山地,地形崎岖,交通闭塞,不利于人们交往,久而久之,便形成各具地方特色的“土话”。
文化是在环境的背景下通过人类的社会活动形成的。环境对文化起着非常重要的作用,人类的社会活动和生产关系也同样起着重要作用。片面强调环境的作用和忽视环境的作用都是错误的。重视环境对文化的影响,有利于因地制宜进行生产建设,继承传统文化,保护文化的多样性,也有利于文化和旅游事业的发展。

5/8 地理课 - 产业结构及其变化

产业结构的概念

指国民经济中的 三大产业 之间的 联系 和 比例 关系。

发达国家的第一产业占比重普遍比较低，第三产业所占比重普遍比较高；而发展中国家反之

产业结构升级

指随着 生产力 的发展，在产业结构中，第二、第三产业相对于第一产业，第三产业相对于第一、第二产业，产业 比重升高的过程。

产业结构变化的一般趋势

  • 第一产业比重持续下降
  • 第二产业比重一般先上升,后下降;
  • 第三产业比重持续上升。

产业结构优化

通过市场机制和政府的政策调控, 使一个国家或地区的三大产业之间保持合理比例关系, 形成本地区的主导产业和支柱产业, 使产业结构与区域内的自然环境、社会经济环境以及生产力发展水平相协调, 体现出有别于其他区域的特点和优势, 以取得最佳的经济效益、社会效益和生态效益。

5/7 地理课 - 不平衡的国际贸易

概念

世界各国之间的商品、劳务和其他生产要素的交换活动,是世界各国在国际分工的基础上通过国际市场进行的交易活动。

发展意义

  • 互通有无,为生产发展提供必要条件
  • 扬长避短,创造更高的经济效益
日本：贸易立国，加工贸易型经济

区位条件

  • 地理位置
  • 自然资源、农产品
  • 经济发展水平

地区差异

  • 国际贸易发展水平差异巨大
    居世界前列的大多为发达国家
国家 进口产品 出口产品
发展中国家 高附加值工业制成品 初级产品、低附制成品
发达国家 初级产品、低附加值制成品 高附加值工业制成品
差异大、不等价
我国出口商品由低附加值的初级产品、工业制成品逐步向高附加值的工业制成品转变
  • 贸易壁垒：关税壁垒、非关税壁垒（如技术壁垒、绿色壁垒等）
    • 2001年12月11日,中国正式加入WTO,成为其第143个成员。
  • 国际贸易运输的主要方式：海洋运输

海洋运输

  1. 建设海港，特别是深水港
    • 上海港的区位优势
      • 位于大陆海岸线中部，长江入海口附近
      • 港阔水深，海陆联运
      • 港口腹地是我国经济最发达的长江三角洲
    • 影响港口经济价值大小的因素：
      • 港口的地理位置
      • 港口的自然条件
      • 港口腹地的大小及其经济发展程度
  2. 采用大型船舶和集装箱运输
    • 集装箱运输的优点
      • 运输量大、集中装卸、车船联运、提高效率
      • 节省包装、保证安全、降低成本

课堂小结

  1. 概念及意义：世界各国之间的商品、劳务和其他生产要素的交换活动。
  2. 区位条件：地理位置、自然资源以及经济发展水平等多种因素会综合影响国际贸易在发展程度与商品结构等方面的地区差异。
  3. 地区差异：各国发展水平差异巨大世界贸易市场大多集中在发达国家; 发达国家与发展中国家的国际贸易商品结构差异很大; 国家间因贸易壁垒、贸易摩擦发生的冲突和纠纷主要通过国际间的贸易谈判解决。
  4. 运输方式：
    • 80%的货运量通过海洋运输完成
    • 发展海洋运输的两个关键
      • 建设海港，特别是深水港
      • 采用大型船舶和集装箱运输

5/7 化学课 - 影响化学平衡移动的因素

化学平衡的移动

概念：可逆发硬中旧化学平衡破坏、新化学平衡建立的过程叫做化学平衡的移动

$$
影响化学平衡的因素
\begin{cases}
温度
\
浓度
\
压强
\end{cases}
$$

浓度对化学平衡移动的影响

浓度变化总结

在其他条件不变时：

  • 增大反应物的浓度，化学平衡向正反应方向移动
  • 减小生成物的浓度，化学平衡向正反应方向移动
  • 增大生成物的浓度，化学平衡向逆反应方向移动
  • 减小反应物的浓度，化学平衡向逆反应方向移动
必须是液体或气体

压强对化学平衡移动的影响

压强变化总结

在其他条件不变时：

  • 增大压强，化学平衡向缩小气体体积的方向移动
  • 减小压强，化学平衡向增大气体体积的方向移动

但有些反应，反应前后气态物质的总体积没有变化，增大压强后减小压强都不能使化学平衡移动，如：

$$\ce {2HI(g) <=> H2(g) + I2(g)}$$

如果平衡混合物都是固体或液体，改变压强也不能是化学平衡发生移动。

温度对化学平衡移动的影响

结论：

  • 温度对化学平衡的影响与化学反应的热效应有关。
  • 在其他条件不变时，升高温度，化学平衡 向吸热方向移动
  • 在其他条件不变时，降低温度，化学平衡 向放热方向移动
温度对吸热反应方向影响比较大

催化剂对化学平衡的影响

化学平衡不能使化学平衡发生移动
但是催化剂能够大大缩短化学反应的时间

关于充入气体的讨论

我们考察以下反应：

$$\ce {N2(g) + 3H2(g) <=> 2NH3(g) + Q}$$

  1. T、V不变，充入$\ce {N2}$，反应物浓度增加，正反应速率增加，逆反应速率随之增加，反应朝正反应方向移动
  2. T、V不变，充入$\ce {He}$，反应物浓度不变，正反应速率不变，逆反应速率也不变，化学平衡不移动
  3. T、P不变，充入$\ce {He}$，压强降低，正逆反应速率均减小，平衡朝增加压强的方向移动，即朝逆反应方向移动
    注：这里勒沙特列原理不能解释
    根据勒沙特列原理，充入氮气，会使得氮气的转化率降低，氢气的转化率升高。这个原理广泛应用于工业生产当中。
